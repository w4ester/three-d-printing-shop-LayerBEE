# Week 4: Spiral Vase (Vase Mode)

Master vase mode with a beautiful spiral vase - fast to print, stunning results!

## What You'll Need
- PLA filament (silk or gradient colors work great!)
- Your 3D printer leveled and ready
- Optional: 0.6mm nozzle for stronger walls

## Recommended Settings
| Setting | Value |
|---------|-------|
| Vase Mode | ON (Spiralize) |
| Layer Height | 0.2-0.3mm |
| Line Width | 0.5-0.6mm |
| Nozzle Temp | 200-215C |
| Bed Temp | 60C |
| Speed | 40-60mm/s |

## Download Options
- [Printables - Low Poly Vase](https://www.printables.com/model/37038)
- [Thingiverse - Twisted Vase](https://www.thingiverse.com/thing:570288)

## How to Enable Vase Mode

**In Cura:**
1. Search "Spiralize" in settings
2. Enable "Spiralize Outer Contour"

**In PrusaSlicer:**
1. Print Settings → Layers and perimeters
2. Check "Spiral vase"

**In OrcaSlicer:**
1. Process → Special mode
2. Enable "Spiral vase"

## Success Criteria
- Smooth, seamless walls (no layer seam visible)
- Consistent wall thickness
- No gaps or holes
- Stable base

## Pro Tips
- Silk filaments look amazing - light passes through!
- Slow down for taller vases (stability)
- Wider nozzle = stronger walls
- These make great gifts!
