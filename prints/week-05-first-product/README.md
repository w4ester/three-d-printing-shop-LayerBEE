# Week 5: Your First Product

The capstone project! Create a product you could actually sell.

## The Challenge

Create a complete product ready for sale, including:
1. A useful 3D printed item
2. Calculated pricing
3. Professional photos
4. Product description

## Suggested Products

| Product | Difficulty | Customizable |
|---------|-----------|--------------|
| Personalized Keychain | Easy | Names/initials |
| Phone Stand | Easy | Colors |
| Desk Organizer | Medium | Sizes/compartments |
| Dice Tower | Medium | Themes |
| Decorative Planter | Medium | Patterns |

## What to Document

### 1. Product Info
- What is it?
- Who is it for?
- What problem does it solve?

### 2. Print Details
- Filament used (type, brand, color)
- Print time
- Settings used
- Any post-processing?

### 3. Pricing Breakdown
```
Materials:    ___ g × $___/g    = $___
Active Time:  ___ min @ $___ /hr = $___
Print Time:   ___ hrs @ $___/hr  = $___
Post-process: ___ min @ $___ /hr = $___
Overhead:     15%                = $___
─────────────────────────────────────
COST:                            = $___
+ Profit (____%):                = $___
─────────────────────────────────────
FINAL PRICE:                     = $___
```

### 4. Photos Required
- [ ] Front view on white background
- [ ] Detail shot (close-up)
- [ ] Lifestyle shot (in use)
- [ ] Scale reference (coin or ruler)

### 5. Product Description
Write 2-3 sentences that would appear in a store listing.

## Share Your Product!
1. Complete all documentation above
2. Take professional photos
3. Fork this repo and add to `community-prints/products/`
4. You're officially a maker-entrepreneur!

Happy selling! - LayerBEE
